# Strokestyles: Interactive demos

This repository contains interactive demos for the Strokestyles project.
Currently the project contains:
- [strokestyles_testbed](https://github.com/autograff-devel/strokestyles_interactive_demos/tree/main/strokestyles_testbed): A "sandbox" that contains multiple demos that demonstrate different aspects of the system. 

To simplify dependency management, it is suggested clone this repository at the same level as the other repositories of the strokestyles echosystem. Global installation instructions for the whole system can be found [here](https://github.com/autograff-devel/strokestyles/blob/master/INSTALL.md).
